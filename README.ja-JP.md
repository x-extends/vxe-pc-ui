👉 [UIドキュメント](https://vxeui.com)  
👉 [テーブルドキュメント](https://vxetable.cn)  

### テーブルとUIの使用

```shell
npm install vxe-pc-ui
```

```javascript
// ...
import VxeUI from 'vxe-pc-ui'
import 'vxe-pc-ui/lib/style.css'
// ...

createApp(App).use(VxeUI).mount('#app')
```

## プロジェクトの実行

依存関係をインストールする

```shell
npm install
```

ローカルデバッグを開始する

```shell
npm run serve
```

コンパイルパッケージング、生成されたコンパイルディレクトリ: es,lib

```shell
npm run lib
```
